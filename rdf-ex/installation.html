# Installation

The RDF.ex Hex package can be installed as usual, by adding rdf to your list of dependencies in mix.exs:

def deps do
  [
    {:rdf, "~> 0.9"}
  ]
end

If you want to read or write JSON-LD, you also have to add json_ld to your dependencies for the separate JSON-LD.ex package. If you want to read or write RDF-XML, you have to add rdf_xml to your dependencies for the separate RDF-XML.ex package.

def deps do
  [
    {:rdf, "~> 0.9"},
    {:rdf_xml, "~> 0.1"},
    {:json_ld, "~> 0.3"},
  ]
end
